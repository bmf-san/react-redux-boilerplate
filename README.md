# React Redux Boilerplate
This is a SPA boilerplate working with react, react-redux, react-router, and so on.

# Features
+ react
+ redux
+ react-redux
+ react-router
+ redux-promise
+ es6
+ webpack

# Get Started
`npm install`
`npm run build`
`npm run start`
